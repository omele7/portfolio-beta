Proyecto personal: Salvando vidas del Covid-19

Aplicación realizada con C# y C++ en Visual Studio 2022 como trabajo final; este trabajo trata sobre un videojuego donde tomamos el papel del doctor que debe eliminar a los virus de un hospital. La principal motivación del proyecto fue la ODS número 3: Salud y bienestar. El objetivo de este trabajo es concientizar, de manera creativa y dinámica, la importancia de mantenernos con buena salud.

Link del repositorio de GitHub: KillTheVirus
